# Faro Travel Guide

## Project Overview
This website provides users with useful information about Faro, Portugal.
It helps users plan their trip by offering details about activities, food, and attractions.

## User Value
Users can easily explore what to do and where to eat in Faro.

## Site Owner Goal
The website promotes Faro as a travel destination and encourages tourism.

## Features
- Navigation menu across all pages
- Responsive layout
- Structured information sections

## Technologies Used
- HTML5
- CSS3

## Deployment
This project is deployed using GitHub Pages.
